Para garantir a integridade e segurança do projeto antes do deploy na AWS, implementei políticas rígidas de versionamento. Entendendo o ciclo de vida do Git (Untracked -> Staged -> Committed), assegurei a configuração prévia do .gitignore para bloquear qualquer injeção acidental de credenciais (.env, .aws) no repositório público, mitigando riscos de vazamento de chaves de acesso cloud.
